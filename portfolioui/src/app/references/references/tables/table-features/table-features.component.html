IMPLEMENTED FEATURES
  1. Data Manipulation
    • Modifying GET API call data to populate mat-table's MatTableDataSource
    • Using datepipe and currencypipe to transform date and price
    • Using slice to delete rows and then reassigning the dataSource's data
  2. Table Style
    • Using .mat-column-columnName to set fixed width for columns
    • Changing th and td color
    • Changing checkboxes color for normal, indeterminate & ripple effect
  3. Pagination, Sorting & Filtering
    • Using MatPagination and dataSource.paginate() method to paginate data
    • Using MatSort and dataSource.sort() method to sort data
    • Using dataSource.filter() method to filter data
  4. Selection & Footer Rows
    • Using SelectionModel to select rows via checkboxes
    • Using mat-footer-cell to calculate total price of selected rows
